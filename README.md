# How to build an MCP server - Calculator Example

⚠️ work in progress ⚠️

## Setup

- Install [direnv](https://github.com/direnv/direnv)
- Install [nix](https://nixos.org/download/)
- Clone this repository
- cd into the directory
- Run `direnv allow`
- Run `pytest`
